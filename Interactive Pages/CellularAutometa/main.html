Alle programmer er skrevet af mig, Noah Dirksen, og kan findes på min GitHub. Alt tekst er skrevet af mig og alle matematiske udledninger er lavet af mig. Da mange fagord går lidt tabt på dansk har jeg skrevet denne artikel på engelsk. Hvis du har svært ved at læse engelsk kan du altid bare lege rundt og se hvordan de forskellige algoritmer virker.
Denne artikel opdaterer automatisk og alt udvalgt kode bliver hentet fra min GitHub når hjemmesiden åbnes. Den nedenstående indholdsfortegnelse er auto genereret og alt udvalgt kode kan udvides på "View source..."-knappen. Så kan man se hele scriptet.
